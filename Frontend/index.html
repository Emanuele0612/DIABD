Welcome to AnimeRecommender

The main goal of this project is to deliver a simple and intuitive graphical interface that enables users to interact with data on users, anime, recommendations and similarities derived from this dataset. This interface offers predefined database query functionalities, so that the user doesn't need to manually write Cypher queries.
